Engenharia de prompt seria a prática de escrever prompts claras, estratégicos e bem estruturados para obter respostas úteis.
LLMs responde com base em probabilidade e contexto textual.
ELEMENTOS DE UM PROMPT
1 - Papel do modelo - Definir quem o modelo deve fingir ser, qual é a persona dele.
2 - Tarefa clara - Explicar exatamente o que deve ser feito.
3 - Formato desejado - Indicar com a resposta deve ser dada
4 - Dar contexto - Fornecer informacoes relevantes para o entendimento do problema.
5 - Restricoes

TIPOS DE PROMPT
Direct prompt - Apenas instrucoes para o modelo, no máximo adiciona um papel para ele. Bom para 
tarefas simples e objetivas. Modelo ja conhece o assunto
One shot / few shot promptinh: fornece modelo alguns exemplos claras e descritivas do comportamento
que o modelo deve seguir.
CHain of thought - Usado em tarefas complexas.
Voce instrui o modelo a pensar passo a passom explicando o raciocinio.
Role prompting - voce define o papel especifico que o modelo deve ser na producao da resposta.

BOAS PRATICAS:
1 - Seja claro e especifico.
2 - Defina o papel
3 - DEfina o formato de resposta
4 - Forneca um bom contexto./
