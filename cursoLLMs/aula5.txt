principais parametros do gpt
parametro model: testar com varios modelos.
janela de contexto: cada modelo tem quantidade maximo de input tokens e output token, isso e dado nas cdocumentacos por context windows
Message: troca de message entre usuario e o gpt
    1 - persona (system) papel que o gpt vai assumir
    2 - toda resposta que o modelo for dar vai ser baseado na persona.
MaxTokens:  especifica o numero maximo de tokens que o modelo na qual deve gerar na reposta.
max_token quebra a resposta, nao se preocupa com a resposta.

temperature e seed:
temperature: ajusta o nivel de criatividade e variacao, na openai varia de 0 a 2, 0 é bom para melhores precisoes. sd
seed controla a aleatoriedade o ideal é repetir o seed para ele controlar isso, nao importa o numerio
